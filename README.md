# Movies Sharing App

**Demo**: https://movies-sharing-app-frontend.vercel.app/

## Technologies:

### Client:

- NextJS
- TypeScript
- Tailwind
- Youtube API
- fetch
- Jest for unit tests

### Server:

- NodeJS
- ExpressJS
- RESTful APIs
- MVC
- AWS RDS - MySQL
- Sinon + Chai for unit tests (controllers, middlewares,...)
