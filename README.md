# Movies Sharing App

**Demo**: https://movies-sharing-app-frontend.vercel.app/

## Technologies:

### Client:

- NextJS
- TypeScript
- Tailwind: styling
- Youtube API: get Youtube video's title and description with video ID
- fetch: API requests
- Jest for unit tests
- Vercel: deployment

### Server:

- JavaScript
- NodeJS
- ExpressJS
- RESTful APIs
- MVC
- AWS RDS - MySQL
- Sinon + Chai for unit tests (controllers, middlewares,...)
- Vercel: deployment
